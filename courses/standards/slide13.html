W3C Web Accessibility Initiative (WAI)

W3C Web Accessibility Initiative (WAI)

Importance

  • Develop web standards that support accessibility by people with disabilities.
  • Promote the accessibility of web based content, authoring tools and web browsers.
  • Raise awareness of how people with various disabilities access and interact with web content.
