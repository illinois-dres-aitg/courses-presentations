Interoperability for Accessibility

Old Model for Accessibility

If it worked with some version of the JAWS screen reader and a particular version of Microsoft Internet Explorer, a website was consideed accessible.

New Model for Accessibility

  • W3C Accessible Rich Internet Application Specficiation
  • HTML5
  • W3C Accessible API Mapping Specification

Screen Readers

  • MacOS and VocieOver: Safari, Chrome, Firefox
  • iOS and VocieOver: Safari
  • Android and TalkBack: chrome
  • Windows 10 and NVDA: Chrome and Firefox
  • Windows 10 and JAWS: Chrome and Firefox
  • Windows 10 and Narrator: Chrome and Firefox

Magnifiers and High Contrast

  • MacOS and Zoom: Safari, Chrome, Firefox
  • iOS and Zoom: Safari
  • Android and Magnifer: chrome
  • Windows 10 and Magnifier: Chrome and Firefox
  • Windows 10 and Zoomtext: Chrome and Firefox

Other Common Assistive Technologies

  • Voice Recognition: Dragon Naturally Speaking, MacOS Speech Recognition
  • Learning Disabilities: Kurzweil 3000, Read and Write Gold
