Coding Landmarks: Adding Labels

Practice adding labels to landmarks

  1. Login into Web Edit
  2. Copy example from the following Web Edit link
  3. Change title "Coding Landmarks: Adding Labels"
  4. Save example to your account

WebEdit template for adding labels to landmarks

Learning Objectives

  • Idenitfy landmarks that need a label (e.g. accessible name).
  • Creating labels that uniquely identify the content in the landmarks.
  • There are two primary ways to add a label to a landmark:
    • aria-label attribute
    • aria-labelledby attribute
  • Verify labels have been added correctly using a screen reader. In addition to the role names, you should now hear the labels when navigate landmarks on a page.
