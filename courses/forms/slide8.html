Voice Over Form Control Support

Keyboard Command Description
Command+F5 Start Voice Over on MacOS.
Control+Option Voice Over (VO) command.
Control+Option++ Toggle VO key lock on and off.
Control+Option+U Open rotor to navigate content on the web page and use the left arrow or right arrow keys to show the list of lkinks.
VO+Command+J Next form control.
VO+Shift+Command+J Previous form control.

Feature Overview

  • Voice Over allows you to view a list of or navigate sequentially to form controls.
  • When navigating a web page Voice Over announces form control information when the browse or focus cursor enters a form control.
