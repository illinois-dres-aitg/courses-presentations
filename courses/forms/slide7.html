NVDA Form Control Support

Keyboard Command Description
Insert+Space
  • Toggle between focus mode and browse mode.
  • Must be in browse mode for the following commands to work.
Insert+F7 List of form controls.
Tab Next interactive element, including form controls.
Shift+Tab Previous interactive element, including form controls.
B Next form control.
Shift+B Previous form control.
B Next button.
Shift+B Previous button.
X Next checkbox.
Shift+X Previous checkbox.
R Next radio button.
Shift+R Previous rado button.
Insert+F1 Help on current element.

Feature Overview

  • In browse mode you can move sequentially form controls on the page or view a list of form controls.
  • When navigating the content of a web page NVDA announces form control information when the cursor is on a form control.
