JAWS Form Control Support

Keyboard Command Description
Insert+Control+Z
  • Toggle on/off virtual JAWS cursor for reading web pages.
  • Must be in virtual mode for the following commands to work.
Insert+7 List of form controls
Tab Next interactive element, including form controls.
Shift+Tab Previous interactive element, including form controls.
F Next form control.
Shift+F Previous form control.
B Next button.
Shift+B Previous button.
X Next checkbox.
Shift+X Previous checkbox.
Enter (on a textbox) Enter content into textbox.
Insert+F1 Help on current element.

Feature Overview

  • In virtual you can move sequentially through the form controls on a page.
  • When navigating a web page JAWS announces form control information when the virtual or PC cursor is on a form control.
