Coding Landmarks: Adding Labels

Labeling Techniques

  • If a page contains more than one of the same landmark, those landmarks need a label.
  • Labels should uniquely identify the landmark from other landmarks of the same type on the page.
  • There are three ways to add a label to a landmark:
    • aria-label attribute
    • aria-labelledby attribute
    • title attribute
