Simple Skip to Main Content

  • Keyboard only users.
  • Screen readers users.

Features

  • Place link (e.g. <a href="#main">Skip to main content</a>) at the beginning of the page.
  • By default use CSS to place link off screeen and then move the link back on screen when the link receives keyboard focus:
    • Use CSS psuedo :focus and :active
      or
    • Use focus and blur events to show and hide the element.
  • Place <a id="main"></a> element immediately after the element used to define the main landmark.
