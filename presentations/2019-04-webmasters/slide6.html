Heading / Paragraph

  • Headings are promoted to the top of the menu
  • All heading menu items are styled the same to reinforce their use as structural elements
  • Heading menu item labels indicate purpose
  • Based on cursor position in the document, only headings that are in sequence can be selected
  • Paragraph formats: Normal, Preformatted text, Address line
  • Provides a Help menu item

Additional configuration options & features:

  • Allow only one h1 (true or false)
  • No gaps allowed (h1;h2;h4 results in h1, h2, h3, h4)
