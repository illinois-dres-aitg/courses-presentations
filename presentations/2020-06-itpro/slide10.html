Accessible Object Model (AOM)

  • Setting the Role attributes using javascript using DOM element object, for example element.role="tablist"; instead of element.setAttribute("role", "tablist");
  • Setting ARIA properties and states with javascript using DOM element object, for example element.ariaLabel="Name"; instead of element.setAttribute("aria-label", "Name");
  • If you set an ARIA role or attributes with script, you can read it from the DOM element object, for example var label = element.ariaLabel; instead of var label = element.getAttribute("aria-label");
