Workshop Summary

Open Source Freely available for anyone to use and customize to their own needs.
W3C Standards Support HTML5 and Accessible Rich Internet Application (ARIA) 1.0 design techniques to meet WCAG 2.0 Level A and AA requirements.
Support Learning Support learning about accessibility by including meta information in each rule:
  • Definition
  • Purpose
  • HTML and ARIA Techniques
  • Informational Links
Integrated Functional Accessibility Evaluator (FAE) 2.0 and AInspector Sidebar work together to understand accessibility features and issues of a web resource.
Visualize Accessibility Bookmarklets provide a means to "visualize" typically hidden accessibility information.
Widgets ARIA Authoring Practices is a reference on how to make custom interactive widgets accessible using web standards.
Accessible Authoring A11yFirst for CKEditor provides a way to support accessible authoring.
