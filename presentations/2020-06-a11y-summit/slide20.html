Design Pattern: Navigation Menu Options

Technique Keyboard Interaction ARIA Description
Keyboard Enhancement
  • tab
  • left arrow
  • right arrow
  • up arrow
  • down arrow
navigation landmark
  • Primarily tab key navigation.
  • Cursor keys provide enhanced navigation.
  • Uses hierarchical lists of links.
  • Uses ARIA navigation landmarks.
  • Examples:
    • Illinois Department of Human Services
    • University of Illinois Home Page
    • Adobe Mega Menu
Menu Button
  • tab
  • up arrow
  • down arrow
Menu Button Design Pattern
  • tab moves into and out of each menu button.
  • up arrow and down arrow move between pull down menu items.
  • Uses button, menu and menuitem widget roles.
  • Examples:
    • Functional Accessibility Evaluator 2.0
    • Illinois Center for Innovation in Teaching & Learning
Disclosure
  • tab
  • space/enter
Disclosure Design Pattern
  • tab moves into and out menubar.
  • space or return opens panel of links.
  • tab and cursor keys move focus between disclosed links.
  • Examples:
    • Illinois Admissions
    • Illinois College of Education
